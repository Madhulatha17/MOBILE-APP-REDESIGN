# MOBILE-APP-REDESIGN

## Selected App
WhatsApp

## Identified Pain Point
Users often struggle to quickly locate unread conversations and important contacts.

## Improvements Made

### Smart Search
Allows users to instantly search chats.

### Favorites Section
Provides one-tap access to frequently contacted people.

### Unread Filter
Displays only unread conversations.

### Cleaner Interface
Improved spacing and visual hierarchy.

### Better User Experience
Reduces navigation effort and improves accessibility.

## Technologies Used
- HTML
- CSS
- JavaScript

## Deliverable
Interactive redesign prototype demonstrating improved UI/UX.

## How to Run
Open index.html in any browser.
